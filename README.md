# About
Simple Personality Test web app using shiny package in **R** language

# Usage
Install package:
```r
install.packages("abdupackage")
```

Launch app
```r
# in your working directory
abdupackage::launch_test(".")
```

Sources:
1. https://openpsychometrics.org/tests/IPIP-BFFM/
